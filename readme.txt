Lösung der Probeaufgabe

1. Erstellen des Layouts
    Um das Layout responsive zu machen, habe ich columns genutzt, die je nach Bildschirmgröße die Breite der jeweiligen Elemente einstellt,
    sodass auf dem Handy alles untereinader angeordnet und bei größeren Bildschirmen das Design wie angegeben ist. Diese Weise habe ich so
    im Kurs an der HTWK gelernt.
    Eine Größe zwischen Mobil und Deesktop, die ich zuerst geplant hatte, habe ich verworfen, da mit dem Design kein wirklicher Bedarf bestand.
    Der Switch und dessen grundsätzliches Styling stammt von w3s. (https://www.w3schools.com/howto/howto_css_switch.asp)
    Der Link der Datenschutzerklärung führt zu einem leeren HTML Dokument, an dessen Stelle die tatsächliche Erklärung verlinkt werden kann.

2. JavaScript Vollständigkeitskontrolle
    Über JavaScript frage ich die Werte ab und kontrolliere ob die Pflichtfelder gefüllt sind. Wenn nicht, werden die entsprechenden Felder rot markiert
    um sie zu kennzeichnen. Dies gilt auch für den Switch.
